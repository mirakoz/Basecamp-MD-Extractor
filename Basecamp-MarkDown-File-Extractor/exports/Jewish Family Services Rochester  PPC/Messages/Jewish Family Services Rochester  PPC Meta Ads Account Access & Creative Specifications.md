Hi Mike,  
  
We need access to the Meta Business Manager and Meta Ad Account. Could you please add [yeliz@almcorp.com](mailto:yeliz@almcorp.com) as Admin to the Meta Business Manager Account?

Once we have access, we will set up the Meta Pixel. After that, we will proceed with creating the Meta Ads campaigns. We will need creative assets for these campaigns.  
  
Here are the specifications for the Meta Ads creative:  
  
Image:   
Aspect Ratio: (1.91:1) - 1200\*628  
Aspect Ratio: (9:16)  - 1080\*1920  
Aspect Ratio: (1:1)  - 1080\*1080  
Max File Size: 30 MB  
File Type: JPEG or PNG  
  
Video:  
Aspect Ratio: (1.91:1) - 1200\*628 (Horizontal)  
Aspect Ratio: (1:1) - 1080\*1080 (Feed size)  
Aspect Ratio: (9:16) - 1080\*1920 (Story Size)  
Maximum File Size: 4 GB  
File Type: MP4 or MOV  
  
It is recommended that three different aspect ratios be created for each image and video.  
  
Please let us know if you have any questions.  
  
Best,  
Yeliz
