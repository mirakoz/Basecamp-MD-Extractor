**Yigit Gok**   
Hi Cameron,  
  
The Setup includes below;  

* Full Campaigns Setup (Including ad assets, tracking parameters etc.)
* Google Tag Manager setup
* Google Tag Manager Events push to Google Ads & GA4
* Google Ads Conversions setup
* Google Ads account links (Analytics etc.)
* Google Ads Audience setup
* GA4 Events setup

  
When start with new accounts/clients, we follow below steps;  

1. Account Access (Google Ads, GTM, GA4) (If an access is missing, we ask for the accounts to be created)
2. We create campaign structure document and share it. Once approved, we set up campaigns on Google Ads.
3. GTM Code implementation to website (We share the GTM code and ask it to be implemented to the website If it's not already implemented)
4. After receiving access to GTM, we set up conversions, events and pixels for Google Ads and GA4.
5. Once all campaigns and tracking are set up, we ask for final approval to launch the campaigns.

  
All the best,  
Yigit
