**Can Basaloglu**   
Hi Kevin,  
  
We have completed the second week on Google Ads. The campaign's targeting is performing well, but it lacks volume. So far, the campaign has generated 41 impressions and one click. We have a couple of recommendations for the campaign:  
  
1. We can keep it as it is. Targeting is good, we have a strong impression share, and most of the time, we are reaching users searching for web design services. We can continue optimizing the campaign and focus on more expensive but highly specific, on-point search terms.  
  
2. We can expand targeting. In addition to our current keywords, such as "web designer near me" or "web design Memphis", we can also target broader terms like "website design" and competitor search terms such as "GoDaddy web design". This is a more aggressive approach compared to the first and current strategy.  
  
3. We can launch a Performance Max campaign to target not only users searching for web design services but also those interested in web hosting, domain registration, web services & development, and people starting a business soon or recently started a business, etc. This will also help boost brand awareness. While it’s a more long-term strategy, given the low volume, it’s worth considering.  
  
Additionally, the attached link includes keyword volume, search terms, auction insights, and the last two days' auction insights to provide more insight into campaign performance as well as an example of Performance Max campaign audience targeting.
