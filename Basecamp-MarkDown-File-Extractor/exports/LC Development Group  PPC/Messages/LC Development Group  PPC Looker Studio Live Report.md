Hi Trevor,

We're excited to share the LC Development Group's dynamic Looker Studio report.

<https://lookerstudio.google.com/reporting/2b707f23-bab6-4e31-ab00-84b91eeecc3d>

Looker Studio is an interactive dashboard that pulls all key performance data into one place, giving us a transparent and real-time view of the campaign's progress. Please take some time to review the report, and please reach out if you have any questions on it. We'd love to schedule a call to walk you through the highlights and answer any questions you may have.

Here is a quick explanation of the report pages:

On the summary page, you'll find a table containing the last 13 months of data that will give us a YoY overview, with some related charts below.

On the overview page, you can choose the dates for the data you want. If you select any campaigns, you can see the results for just that campaign. If you leave the campaign selection dropdown menu blank, you will see data for all campaigns. On this page, you'll also find relevant charts and a table for the conversion actions we are tracking on the account. "All conversions" is a combination of our primary and non-primary conversions, while the "Conversions" column only includes primary conversion actions.

On the detailed report page, you can again choose the dates for the data you want. If you select any campaigns, you can see the results for just that campaign. Here you can find more details about ad groups per campaign, keyword, and search term results, and device and gender segmentation results.

  

Best regards,

Yeliz
