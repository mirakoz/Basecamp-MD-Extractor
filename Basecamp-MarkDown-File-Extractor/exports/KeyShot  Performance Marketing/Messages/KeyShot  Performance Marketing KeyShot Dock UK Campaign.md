### KeyShot Dock UK Campaign - Audience Strategy

Posted by Asger Teglgaard on May 22, 2025

[![Asger Teglgaard](https://asset1.basecamp.com/2325709/people/18906982/photo/avatar.96.gif "Asger Teglgaard")](/2325709/people/18906982)

Hey Mucahit,

I need your assistance on the audience setup for the upcoming UK campaign.

We talked about having a broad UK audience, but we also have our tier one account list that I'd like to see if we can match. The issue is we only have 360 contacts and 11 accounts on that list. You can see it here: <https://app.hubspot.com/contacts/2757257/objects/0-1/views/my/list>

I'm thinking we might need a hybrid approach - broad UK targeting plus whatever we can do with the matched audience from our tier 1 list.

Any ideas on how we can set this up?

Thanks,

Asger

**Mucahit Yilmaz**   
Hi Asger,  
  

Since the list will be pretty narrow once uploaded to LinkedIn, we can consider using Message Ads or Conversation Ads. Message Ads are similar to the ones we tested with the Qbank audience, while Conversation Ads also appear in users’ inboxes but allow for multiple CTAs. These CTAs can either guide users through a more personalized conversation based on our inputs or direct them to a chosen landing page for more product information. I'm not sure if Linkedin'll be able to match that 360 contacts fully, so if we don't receive the impressions we expect, we can move forward with predictive audiences.

We can build a narrow predictive audience (similar to lookalike audiences on Meta) based on the account list and target it with Video View or Thought Leader Ads. These campaign types were our picks for top funnel campaigns when we had the meeting for this UK campaign strategy.

Let us know what you think.

Best,

Mucahit
