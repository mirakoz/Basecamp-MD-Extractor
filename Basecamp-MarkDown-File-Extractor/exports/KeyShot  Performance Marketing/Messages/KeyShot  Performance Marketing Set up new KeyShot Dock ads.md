### Set up new KeyShot Dock ads

Posted by Asger Teglgaard on Oct 21, 2024

[![Asger Teglgaard](https://asset1.basecamp.com/2325709/people/18906982/photo/avatar.96.gif "Asger Teglgaard")](/2325709/people/18906982)

Hi there   
  
We are ready with visuals and copy for the first round of new KeyShot Dock ads on LinkedIn.   
You can find the ad copy here under the "Updated Dock Ads" tab:   
[https://docs.google.com/spreadsheets/d/1QJ5BfY9qNdDSeVQIgW9fpvhqPboTRG1bHvM…](https://docs.google.com/spreadsheets/d/1QJ5BfY9qNdDSeVQIgW9fpvhqPboTRG1bHvMzs9B2ZVE/edit?usp=sharing "https://docs.google.com/spreadsheets/d/1QJ5BfY9qNdDSeVQIgW9fpvhqPboTRG1bHvMzs9B2ZVE/edit?usp=sharing")  
  
Visuals can be found here:   
[https://drive.google.com/drive/folders/1z0tfZWVH8V4uXoYB3EVueJtLvujeAZN6?us…](https://drive.google.com/drive/folders/1z0tfZWVH8V4uXoYB3EVueJtLvujeAZN6?usp=sharing "https://drive.google.com/drive/folders/1z0tfZWVH8V4uXoYB3EVueJtLvujeAZN6?usp=sharing")  
  
The visuals are named accordingly to the ad copy.   
  
I want to test the product ads on a cold audience using the audiences defined under the "Targeting" tab in the sheet. : Nordics + Marketing + Manufactor, UK + Marketing + Manufactor, Benelux + Marketing + Manufactor.   
  
Create a new Campaign for each audience and please let me know if you have any ideas for optimizing the audiences and campaigns.   
  
For retargeting add the trust ads.   
  
**Budget:**I want to start spending most of our PPC budget on these LinkedIn campaigns.   
Currently I have a total monthly budget going forward on about $28k.   
Please set up a budget for nov and dec using all the money we can on LinkedIn while still running our Google Ads.   
  
Regarding budget I want you to add most of the budget into the cold campaigns to begin with.   
Hopefully we can build a bigger and bigger retargeting audience with time.   
  
  
**Note:**I will request more ad visuals for both Cold and retargeting with our content team so we continuously can add more ads into both.
