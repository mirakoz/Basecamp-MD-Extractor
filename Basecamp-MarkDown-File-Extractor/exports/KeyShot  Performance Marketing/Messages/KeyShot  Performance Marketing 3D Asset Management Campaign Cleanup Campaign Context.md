Posted by Asger Teglgaard on Apr 14, 2025

Hey  
  
i've been through the terms for the 3D campaign for KeyShot Dock.   
  
**For context:**  
KeyShot Dock is our Digital Asset Management (DAM) solution that now offers enhanced capabilities for managing 3D assets. This unique value proposition positions us at the intersection of traditional DAM and 3D content management - an emerging need for companies dealing with growing libraries of 3D content.

**What we're trying to achieve with this campaign:**

* Capture organizations looking specifically for DAM solutions that can handle 3D assets
* Target companies with established 3D content workflows seeking better organization and distribution
* Position KeyShot Dock as the specialized solution for enterprise 3D asset management
* Differentiate from general 3D modeling tools, viewers, and creation software

  
**Current Issues with Keyword Strategy**

After reviewing the search terms report from April 1-6, 2025, we've identified that the campaign is attracting significant traffic from users looking for:

1. 3D modeling software
2. 3D viewers and file viewers
3. Generic 3D design tools
4. Educational 3D tools

These users are unlikely to convert as they're looking for creation tools rather than asset management solutions. We need to refocus our keyword strategy to better target the DAM + 3D intersection.

  
**Keyword Optimization Actions
1. Add These Negative Keywords**

Please add the following as negative keywords to prevent spend on irrelevant traffic:

**Exact Match:**

* "3d model viewer"
* "3d object viewer"
* "3d file viewer"
* "online 3d viewer"
* "model file viewer"
* "3d software"
* "3d designer software"
* "create your own 3d model"
* "preview 3d models"
* "3d reference models"
* "pixyz studio"
* "how to make a 3d classroom model online"
* "3d retail store design software"

**Phrase Match:**

* "3d viewer"
* "create 3d"
* "make 3d"
* "design 3d"
* "3d modeling"
* "classroom model"
* "3d printing"
