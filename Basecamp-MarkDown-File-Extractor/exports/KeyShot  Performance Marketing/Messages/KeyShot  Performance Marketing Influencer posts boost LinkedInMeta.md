### Influencer posts boost LinkedIn/Meta

Posted by Vahag on Mar 4

[![Vahag](https://asset1.basecamp.com/2325709/people/19122740/photo/avatar.96.gif "Vahag")](/2325709/people/19122740)

Hey Fathy,  
  
We have some influencers who agreed to have their posts boosted by keyshot. Can you please advise how we can coordinate this on LinkedIn? Do we need to send them some kind of connection, partnership agreement type of thing?   
  
On meta, I think we can do just collaboration. Please confirm.   
  
Please let me know, and I'll provide the profiles and the posts we want to boost, and we can take it from there.

#### Discuss this message

[![Fathy Elsayed](https://asset1.basecamp.com/2325709/people/18538829/photo/avatar.96.gif "Fathy Elsayed")](/2325709/people/18538829)

**Fathy Elsayed**   

Hi Vahag,

For **Meta**, we can run this through **Branded Content / Partnership Ads**.  
  
1st options   
**The influencers as the first identity**. In this case, the influencers will need to tag KeyShot as a *Paid Partnership* and approve us as an advertiser so we can promote the post. this will

2nd option:   
**KeyShot as the first identity**. In this case, we’ll generate a **Partnership Ad code** from our side and send it to the influencer. They will need to use that code when creating the post so we can access the media and run the ad from our Ads Manager while keeping KeyShot as the primary identity.  
  
reference: <https://help.instagram.com/1168446653551639>

For **LinkedIn**, it works a bit differently. To promote their posts directly, the influencers will need to grant our **LinkedIn Ads account permission for Direct Sponsored Content**. Once we have that access, we can select their post in Campaign Manager and run it as a sponsored post, as shown in the attached screenshot.   
More info here: 

Best,  
Fathy
