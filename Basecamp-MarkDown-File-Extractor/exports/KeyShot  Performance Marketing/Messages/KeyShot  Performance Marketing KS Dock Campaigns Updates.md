### KS Dock Campaigns Updates

Posted by Fathy Elsayed on Jan 22, 2025

[![Fathy Elsayed](https://asset1.basecamp.com/2325709/people/18538829/photo/avatar.96.gif "Fathy Elsayed")](/2325709/people/18538829)

Hi Asger, thanks for the Call today, we discussed the below points for the Dock campaigns:   
  
**On LinkedIn:**  

We paused FUNC and LEADER audience traffic campaigns and moved their budget to lead campaigns. If the budget increase doesn't help the lead campaigns we'll suggest moving some of this budget to video campaigns since those have reasonable CPC's compared to traffic campaigns

Paused -- Moved budget to

EU-COLD-FUNC-TRAFFIC -- EU-COLD-FUNC-LEAD

EU-COLD-LEADER-TRAFFIC -- EU-COLD-LEADER-LEAD

NA-COLD-LEADER-TRAFFIC -- NA-COLD-LEADER-LEAD

NA-COLD-FUNC-TRAFFIC -- NA-COLD-FUNC-LEAD

  
**On Google Ads:**  
we will pause all time unperformed campaigns and shift the budget to a new display campaign,  
we will send recommendations for display campaigns (Layout + sizes)  
We will focus more on the top exact matching keywords.   
  
**On Bing ads:**  
We will launch a new 3 search campaign as follows:   
US, UK, and Nordic countries with a $15 daily budget as a start, using the same ad copy from the Google ads campaigns.   
  
We need to set up a website heatmap on the dock landing page to analyze the user experience.   
  
  
Regards,  
Fathy
