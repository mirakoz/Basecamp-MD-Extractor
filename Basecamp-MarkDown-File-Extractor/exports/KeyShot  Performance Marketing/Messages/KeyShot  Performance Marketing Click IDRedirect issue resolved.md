### Click ID/Redirect issue resolved

Posted by Josh Mings on Sep 1, 2025

[![Josh Mings](https://asset1.basecamp.com/2325709/people/18698286/photo/avatar.96.gif "Josh Mings")](/2325709/people/18698286)

Hey guys, there was an issue when a GCLID was stripped from a link that is redirected:   
  
So, something like this:  
[https://www.keyshot.com/keyshot-studio-trial/?\_gl=1\*1m1ngow\*\_up\*MQ..\*\_ga\*MT…](https://www.keyshot.com/keyshot-studio-trial/?_gl=1*1m1ngow*_up*MQ..*_ga*MTIzNDYwMzA1OS4xNzU2NzUwNjcy*_ga_Y0KR9JQZMF*czE3NTY3NTA2NzIkbzEkZzAkdDE3NTY3NTA2NzIkajYwJGwwJGgw "https://www.keyshot.com/keyshot-studio-trial/?_gl=1*1m1ngow*_up*MQ..*_ga*MTIzNDYwMzA1OS4xNzU2NzUwNjcy*_ga_Y0KR9JQZMF*czE3NTY3NTA2NzIkbzEkZzAkdDE3NTY3NTA2NzIkajYwJGwwJGgw")  
  
Would end up like this:   
<https://www.keyshot.com/keyshot-studio-trial/>  
  
It may have affected some of the tracking. I've checked, and the redirects have been removed, and the ID is following through. Let me know if you have any questions or concerns.
