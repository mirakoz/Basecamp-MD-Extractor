Hi Fathy, just a heads up that I re-categorize some of the DPS in Usercentrics. Here's the logic. If a cookie is used:  

* To run the site or deliver a user-requested feature (security, login, CSRF, remembering consent): **Essential**
* To measure site usage / maintain session + visitor identity for analytics & CRM: **Functional/Analytics**
* For marketing-specific, ad targeting/retargeting or ad network measurement: **Marketing/Advertising**

- I'm considering consolidating Marketing & Advertising since they are separated at the moment, but waiting on the final input from legal. If we need to move some functional to marketing, we would keep it, so we can differentiate between what would be marketing/analytics and advertising.
