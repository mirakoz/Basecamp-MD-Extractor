### More structured

Posted by Josh Mings on Aug 24, 2023

[![Josh Mings](https://asset1.basecamp.com/2325709/people/18698286/photo/avatar.96.gif "Josh Mings")](/2325709/people/18698286)

Hi all, Bailey and I have been going over campaigns, events, and tags. It's getting really hard to report on the campaign performance with events in GTM/GA4 that are not very useful. We'd like to go over this and get more structured.   
  
Some things we would like to discuss...  
  
**Planning:**  

* What is tracked as an event
* What is tracked as a conversion event
* Defining flow for event tracking

**Implementation:**  

* Where events are set up
* How the event tags are set up
* Event name nomenclature

**Execution:**  

* Reporting on events
* Campaign views/dashboards
* Optimizing events/flow

  
One example is attached here, where we are tracking branded ad campaign visits to keyshot.com homepage where clicking the 'Free Trial' button is captured as a conversion event. This is not an accurate measure of downloads/activations.   
  
It's more or less tweaks we need to make going forward. We can discuss the past/current campaigns to be better positioned for the next.
