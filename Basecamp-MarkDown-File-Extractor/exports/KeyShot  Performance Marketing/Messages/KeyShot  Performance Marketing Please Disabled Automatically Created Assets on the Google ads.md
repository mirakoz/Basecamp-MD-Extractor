### Please Disabled Automatically Created Assets on the Google ads

Posted by Bailey Liu on Jul 2, 2024

[![Bailey Liu](https://asset1.basecamp.com/2325709/people/18698287/photo/avatar.96.gif "Bailey Liu")](/2325709/people/18698287)

Hi Fathy and Burak,   
  
I remember that I reminded your team several times that ALL the automatically generated assets, contents, snippets, and site links should be disabled. I found several auto-created highlights were created and used in all our campaigns in the Google ads assets. Please double-check it again to disable all of them. Thanks  
  
Bailey

#### Discuss this message

[![Fathy Elsayed](https://asset1.basecamp.com/2325709/people/18538829/photo/avatar.96.gif "Fathy Elsayed")](/2325709/people/18538829)

**Fathy Elsayed**   
Hi Bailey,   
  
We are running two types of campaigns, Google search ads and Pmax ads, for google search as per you can see in the screenshots it's been turned off since the previous discussion, for pmax campaign we found that some campaigns were having the text assets only turned on, and we turned it off now, as far as we can see now text assets were related to Pmax assets headlines/descriptions which are running with the original ad copy we provided.   
  
In the link below you can see the automated created assets performance last month,   
  
[https://docs.google.com/spreadsheets/d/1ASA7LeSOELcgcj-C1ekDNP6jmWVSIjCeCwX…](https://docs.google.com/spreadsheets/d/1ASA7LeSOELcgcj-C1ekDNP6jmWVSIjCeCwXu0b_b4KY/edit?usp=sharing "https://docs.google.com/spreadsheets/d/1ASA7LeSOELcgcj-C1ekDNP6jmWVSIjCeCwXu0b_b4KY/edit?usp=sharing")  
  
only the business logo and business name have been got impressions and clicks, as we are turning it off, even if, we've paused all automated created assets into the Google ads account. - attached screenshot -  
  
Please let us know if you still see the same issue in the Google ads account.   
  
Regards,  
Fathy
