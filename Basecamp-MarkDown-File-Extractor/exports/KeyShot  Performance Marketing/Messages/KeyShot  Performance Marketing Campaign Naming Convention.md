### Campaign Naming Convention

Posted by Bailey Liu on Feb 20, 2025

[![Bailey Liu](https://asset1.basecamp.com/2325709/people/18698287/photo/avatar.96.gif "Bailey Liu")](/2325709/people/18698287)

Hi ALM team,   
  
As we ramped up our campaign activities on both Studio and Dock, the need to create various reports has increased as well. To help us expedite the report building with clarity, Josh has recommended implementing a more structured campaign naming convention across channels. Please see his suggested format below:   
  

[Date: YYYYMM]\_[Campaign Name]\_[Channel]\_[Medium]\_[Product]\_[Content]\_[Description]

202412\_PD2M\_LinkedIn\_Feed\_Dock\_Whitepaper\_PIMDAM     
  
I'd like to have your guys' input on this format and if we can update this to our current active campaigns. Would this affect the campaign running?   
  
Thanks,  
Bailey
