### Recommendations for Green Group Account

Posted by Buse on Aug 15, 2024

[![Buse](https://asset1.basecamp.com/2325709/people/18498332/photo/avatar.96.gif "Buse")](/2325709/people/18498332)

Hi Francisco,

We have compiled some actions that we can take in the Green group account. We need to work together to take these actions. We need some assets. Can you please review and share your ideas? We can quickly take action for the suggestions you accept  
  

**Recommendations For Green Group**

**1. Using Pmax**

  

It is one of the structures we especially recommend for brands with limited budgets. We can use many campaign types within the Pmax campaign. We had previously decided to do display ads. We can create our display ads within Pmax without distributing campaign budgets too much. In addition, if you prepare a video, we will use these videos as a campaign type within Pmax. Apart from this, in search ads, we currently only appear in front of people searching for the keywords we have targeted. In Pmax, the website is dynamically scanned and the keywords are automatically determined. In this way, we also lose search terms. We need the following assets for this.

  

*Creative for Display*

  

1. Landscape Image: 1200x628

2. Square Image: 1200x1200

3. Portrait Image: 960x1200

4. Square Logo: 1200x1200

5. Landscape Logo: 1200x300

  

*Video Asset*

  

We need a Youtube video link.

  
  

![](https://lh7-rt.googleusercontent.com/docsz/AD_4nXd7J_inm6rNbl66ilb1z7VlRq4Nwn-eSSdFth9JZ45ntmL8Xa2PlxV3o_kUOmT1QVVPCrZghyxAq0gtXy5Tpo3xQhewkMOY-S506-pSxrqLhDJ4wwU2xEUvQ4AV1iyat_az6YT6mWyMQ551AO-duv940Eyo?key=O-KG3O2TjaO0bE_mrsOCoQ)

**2. Customer List**

  

The user data you have within your brand is quite efficient in advertisements. This data can be used both for remarketing and to find similar audiences. We can use this data to feed the Pmax campaign, which we mentioned above. In this way, we can ensure that the campaign works more efficiently.

**3.****R****emarketing / Budget Increase**

  

We have created many remarketing audiences on our account. We need a budget increase to be able to use these audiences. If you want to allocate test budgets, we can carry out remarketing studies on these audiences. While using Pmax's video campaigns to specifically increase these audiences, we can also directly target these people on different campaigns. We can shorten the persuasion processes with special visuals. Or, we can prepare e-mail advertisements that focus on the benefit points that will enable them to fill out forms.

  
Best Regards,  
Buse
