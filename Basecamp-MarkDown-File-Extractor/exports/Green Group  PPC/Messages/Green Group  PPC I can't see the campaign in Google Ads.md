**Can Basaloglu**   
Hi Francisco,  
  

There’s no issue with the ads. The limited serving notice appears due to the Consumer Finance policy, which restricts the use of demographic targeting and ZIP code based location targeting. However, since we’re not using either of these, it doesn’t affect our campaigns.

As for the issue, you can try disabling any ad blockers if you’re using one. Clearing cookies may also help, though you’ll need to log in again afterward. If neither works, try using incognito mode and disabling all other extensions there.

Please let me know if that helps. Alternatively, you can access our Looker Studio report, which is directly integrated with Google Ads, so the data will match exactly.
