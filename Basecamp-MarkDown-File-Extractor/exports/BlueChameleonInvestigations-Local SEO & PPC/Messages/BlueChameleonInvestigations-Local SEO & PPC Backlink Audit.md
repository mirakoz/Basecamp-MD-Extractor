Hi Luciana,

We conducted a backlink audit on the website to identify any toxic backlinks that could negatively impact its search engine performance.

Please find the attached backlink audit report for the website.

Key Findings:

* Overall Toxicity Score: 38
* Total Backlinks Analyzed: 416
* Referring Domains: 159

Breakdown:

* 66 domains are non-toxic
* 46 domains are potentially toxic
* 38 domains are confirmed toxic

A “Toxic Link” is a link identified by the backlink tool as being potentially harmful to your site, like Spammy Links, Irrelevant Links, Links from Penalized Sites, etc. We highly recommend either getting rid of these backlinks or disavowing these Links in Google Search Console (GSC) so Google can ignore these toxic links when evaluating your website. Getting rid of these backlinks is a difficult and time-consuming process, as it will require contacting and requesting every webmaster to remove our link, which is not always successful. The easy way is to submit a disavow list in GSC simply.

  

Please review these links and let us know if you have any questions.
