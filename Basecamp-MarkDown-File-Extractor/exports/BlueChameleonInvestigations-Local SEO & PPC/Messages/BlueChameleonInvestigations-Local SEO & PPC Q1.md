Hi Luciana,

  

We conducted a backlink audit on the website to identify any toxic backlinks that could hurt the performance of the website in search engines.

  

Please find attached the backlink audit report for this domain.

  

A “Toxic Link” is a link identified by the backlink tool as being potentially harmful to your site, like Spammy Links, Irrelevant Links, Links from Penalized Sites, etc. We highly recommend either getting rid of these backlinks or disavowing these Links in Google Search Console (GSC) so Google can ignore these toxic links when evaluating your website. Getting rid of these backlinks is a difficult and time-consuming process as it will require contacting and requesting every webmaster to remove our link, which is not always successful. The easy way is to simply submit a disavow list in GSC.

  

Please review these toxic links and confirm that none of them are created or built by the client. We will then get these disavowed in Google.

  

Please let us know if you have any questions.

  

Best Regards,
